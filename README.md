#Página de Muestra 



## Descripción 
    Pagina web con página principal index.html mas
    tres paginas secundarias (contacto, actividades, ubicación, ).

    Menu de navegación, Logo con enlace a pagina principal, menu respopnsive,
    formulario de contacto que envía correo al cliente, enlace a numero de whatsapp del
    cliente.

    Uso de Framework Bootstrap.

    Pagina Responsive

    Tecnologías aplicadas: HTML5 - CSS - JS.
